# 🌾 Agro Forte & Sustentável

Uma página web moderna e responsiva dedicada a promover a união entre a força do agronegócio e as práticas de sustentabilidade ambiental. Este projeto foi desenvolvido para demonstrar como a tecnologia e o manejo consciente podem andar de mãos dadas para o futuro do planeta.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais, mantendo o código limpo, leve e de fácil manutenção:

* *HTML5:* Estruturação semântica de todo o conteúdo da página.
* *CSS3:* Estilização personalizada com design moderno, uso de Flexbox para responsividade, gradientes e transições suaves.
* *Google Fonts / Imagens Unsplash:* Recursos visuais de alta qualidade voltados para a temática ambiental.

## 🌿 Funcionalidades e Destaques

* *Design Responsivo:* Se adapta bem a telas de computadores, tablets e celulares.
* *Identidade Visual Temática:* Uso de cores que remetem à natureza (verde) e à força da terra (laranja).
* *Seções Informativas:* Divisão clara entre os pilares da sustentabilidade (Manejo, Água e Energia) e a filosofia do Agro Verde.
* *Menu Fixo (Sticky):* Facilita a navegação do usuário enquanto ele rola a página.
